Faster Algorithms Using AI

On June 7, DeepMind AI team published a paper in which they use AI to invent faster algorithms.

The method for doing that is based on AlphaZero, an AI system for playing board games like chess and Go.
They treat the problem of building a program like a game, where the next move is the next program instruction.

AlphaDev operated at the level of assembly instructions and uses deep reinforcement learning.

One of the algorithms this system invented is a faster algorithm for sorting, which is integrated into C++ standard library!

The reward used for training this system is how many items in the lists sorted correctly, also they include a reward related to speed, on the basis of either the total number of instructions or the processing time.
